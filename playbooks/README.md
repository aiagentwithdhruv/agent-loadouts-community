# Playbooks — Ready-Made AI Systems for Real Work

Playbooks are **complete AI setups** for specific use cases. Each one tells you exactly what skills, context, and tools to combine — so you can go from zero to working system in minutes.

## How to Use a Playbook

1. Pick the playbook that matches your role
2. Copy the files it lists into your project
3. Fill in your specific details (name, business, goals)
4. Open with Claude Code / Cursor / any AI tool
5. The AI instantly becomes your domain expert

---

## Create Your Own Playbook

A playbook is just a document that lists:
- Which context files to create
- Which skills to include
- Which tools to set up
- A step-by-step setup guide

See the [Freelancer playbook](freelancer.md) as a reference.
