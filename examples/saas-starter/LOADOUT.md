# SaaS Starter — Agent Loadout

> A starter loadout for any SaaS company. Customize with your product details, market research, and sales playbook. Your AI agent will understand your business from day one.

---

## What's Included

| File | Type | Purpose |
|------|------|---------|
| `.claude/CLAUDE.md` | Context | Auto-loaded master context |
| `knowledge/saas-metrics.md` | Knowledge | SaaS metrics and benchmarks |
| `runbooks/deploy.md` | Runbook | Generic deployment checklist |
| `tests/smoke-tests.md` | Tests | Basic smoke tests |

---

## Quick Start

1. Copy this folder to your project root
2. Edit `.claude/CLAUDE.md` with your company details
3. Add domain-specific knowledge files
4. Open with Claude Code — agent auto-loads your context
