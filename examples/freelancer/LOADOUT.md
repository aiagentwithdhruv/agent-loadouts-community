# Freelancer — Agent Loadout

> Equip your AI agent with everything it needs to help you run a freelance AI development business. Services, pricing, proposal templates, client management, and outreach strategies.

---

## What's Included

| File | Type | Purpose |
|------|------|---------|
| `.claude/CLAUDE.md` | Context | Auto-loaded context — services, pricing, processes |
| `knowledge/pricing-guide.md` | Knowledge | Market rates for AI/automation services |
| `runbooks/new-client.md` | Runbook | Client onboarding checklist |

---

## Quick Start

1. Copy this folder to your project root
2. Edit `.claude/CLAUDE.md` with your services, rates, and portfolio
3. Open with Claude Code — agent can draft proposals, manage clients, suggest pricing
